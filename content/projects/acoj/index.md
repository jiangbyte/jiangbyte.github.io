An online judge system that supports multi-language code execution via sandboxed judging, AI-assisted problem-solving, and a full admin panel for problem management.

## Components

- **acoj** — Backend API written in Go, built on hei-gin framework, integrates go-judge sandbox for secure code execution
- **acoj-admin** — Admin dashboard built with Vue 3 + TypeScript + Ant Design Vue
- **acoj-pc** — User-facing frontend built with Vue 3

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Go, hei-gin, go-judge |
| Admin UI | Vue 3, TypeScript, Ant Design Vue |
| User UI | Vue 3 |
