A full-stack rapid development framework designed for building admin panels and API services efficiently, available in Go, Python, and Vue implementations.

## Components

- **hei-gin** — Backend scaffolding in Go + Gin framework
- **hei-fastapi** — Backend scaffolding in Python + FastAPI
- **hei-admin-vue** — Admin dashboard built with Vue 3, Vite, TypeScript, and Ant Design Vue
- **hei-pc-vue** — User-facing frontend built with Vue 3

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Go Backend | Go, Gin |
| Python Backend | Python, FastAPI, uvicorn |
| Admin UI | Vue 3, Vite, TypeScript, Ant Design Vue |
| User UI | Vue 3 |
