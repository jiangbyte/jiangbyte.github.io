## Getting Started

Hugo is a fast and modern static site generator written in Go. It takes Markdown files and renders them into a complete static HTML site.

### Installation

You can install Hugo via package managers:

```bash
brew install hugo       # macOS
sudo apt install hugo   # Ubuntu
```

### Configuration

The `hugo.yaml` file contains all site settings like base URL, title, theme, and language. Theme-specific settings go in `params.yaml`.

## Writing Posts

Posts are written in Markdown with frontmatter (YAML between `---` lines) for metadata like title, date, categories, and tags.

```markdown
---
title: My Post
date: 2025-01-01
categories:
  - Tech
tags:
  - hugo
---

Content goes here...
```

## Features

- **Masonry layout** for the post grid
- **Light/Dark mode** toggle
- **Client-side search** powered by Fuse.js
- **Table of contents** sidebar on single posts
- **RSS feed** for subscribers
