## Why Rust?

Memory safety without a garbage collector. Zero-cost abstractions. Amazing tooling with Cargo.

## First Impressions

The borrow checker feels strict at first, but it catches bugs at compile time that would be runtime crashes in other languages.

```rust
fn main() {
    println!("Hello, Rust!");
}
```
