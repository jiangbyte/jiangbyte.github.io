A collection of tools that make my development workflow smoother.

<!--more-->

## Editor

VS Code with Vim keybindings. Lightweight, extensible, and great IntelliSense.

## Terminal

- **Kitty** - GPU-accelerated terminal emulator
- **tmux** - terminal multiplexer
- **zsh** + **starship** - fast shell with a beautiful prompt

## Git

- lazygit - terminal UI for git
- gh - GitHub CLI for PRs and issues

## Container

Docker + docker-compose for local development environments.
